# FavoriteFiles

This is a simple plugin that was created to save favorite files that are not part of a project.  It can also be used to
save and load groups of files which may be useful even for files part of a project.

# Features

-   Add/remove/open file(s) in favorites.
-   Optionally store files in groups and open entire groups.
-   Toggle project specific favorites.
